## Summary

The persistent storage system ensures:

1. **All user data flows through PostgreSQL**
   - XP, levels, LC, inventory all stored in database
   - Real-time writes on every change
   - No in-memory caching of user state

2. **Commands always fetch fresh data**
   - !niveau â†’ db.getUser() â†’ PostgreSQL
   - !sac â†’ db.getInventory() â†’ PostgreSQL
   - !lc â†’ db.getUser() â†’ PostgreSQL

3. **Data persists across restarts**
   - Bot restart = memory cleared
   - Database = unchanged
   - Next command = fetches from database
   - Result = all data preserved âœ…

4. **Automatic user creation**
   - First interaction â†’ Check if user exists
   - If not exists â†’ Create with defaults
   - All future operations â†’ Use existing record
